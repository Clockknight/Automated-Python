
Q:

1. What is a relative path relative to?

A:

Relative to the current working directive

Q:

2. What does an absolute path start with?

A:

It begins with the root folder

Q:

3. What do the os.getcwd() and os.chdir() functions do?

A:

Find the current operatiting directive, and change the directive respectively.

Q:

4. What are the . and .. folders?

A:

"This directory" and "The parent folder", respectively

Q:

5. In C:\bacon\eggs\spam.txt, which part is the dir name, and which part is the base name?

A:

C:\bacon\eggs and spam.txt respectively

7. What happens if an existing file is opened in write mode?

A:

It is erased, and it is written over from the start.

Q:

8. What is the difference between the read() and readlines() methods?

A:

Read() will return the entire text file, while readlines() will return a list of strings, one object for each line.
